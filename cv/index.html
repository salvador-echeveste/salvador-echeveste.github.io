Projects

  • 2024.01 - 2025.01
    Heteroscedastic Multi-Objective Optimization Platform (Hip Exoskeleton)
    Developed heteroscedastic Gaussian Processes with ARD Matérn 5/2 kernel and EHVI for Pareto exploration in multi-objective Bayesian optimization with input-dependent noise
    • 23-31% RMSE reduction vs homoscedastic baselines
    • 10× faster convergence (12 min vs 2-4 hrs)
    • 18.2% EMG and 17.2% metabolic cost reductions in 11-subject studies
  • 2024.01 - 2025.01
    Real-Time EMG Signal Enhancement
    Designed adaptive filtering algorithms using Hankel trajectory reconstruction with truncated SVD and Bayesian-regularized Koopman regression
    • +108.7% combined metric improvement
    • Decoupled noise from signal (r≈0.011 vs -0.64 baseline)
    • Sub-millisecond latency for real-time processing
  • 2024.01 - 2024.12
    Anthropometric ML for Controller Prediction
    Developed machine learning models using elastic net feature selection and Akima interpolation for optimal controller class prediction from anthropometric features
    • 75% accuracy in controller prediction
    • Reduced Bayesian HIL optimization time by 26.5%
    • Improved controller performance by 9.98% on average
